# entheogen-assets

Asset library for the Entheogen project. This repository stores static assets used across associated projects.

## Directory Structure

| Directory       | Contents                                      |
|-----------------|-----------------------------------------------|
| `logos/`        | Logo files (SVG, PNG, etc.)                   |
| `screenshots/`  | App and feature screenshots                   |
| `promo/`        | Promotional materials (banners, graphics, etc.) |
| `gifs/`         | Animated GIFs (demos, tutorials, etc.)        |

## Usage

Reference assets directly via the raw GitHub URL, for example:

```
https://raw.githubusercontent.com/New-Psychonaut/entheogen-assets/main/logos/<filename>
```
